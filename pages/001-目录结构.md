# 环境搭建

<v-clicks>

新建根目录

</v-clicks>

<v-clicks>

``` bash
mkdir vue & cd vue
```

化繁为简，本文重在学习 Vue，故而采用原生 [`Javascript 模块`](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Guide/Modules)。初始项目结构如下：

```
vue/
  ├── index.html // 示例页
  └── src/ // Vue 源文件
      └── index.js // Vue 主文件
```

</v-clicks>

::right::
